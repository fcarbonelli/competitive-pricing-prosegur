# Logos de Competidores

Agregar los logos de cada competidor en esta carpeta con los siguientes nombres:

- `adt.png` - Logo de ADT
- `verisure.png` - Logo de Verisure
- `protek.png` - Logo de Protek
- `telesentinel.png` - Logo de Telesentinel
- `alarmar.png` - Logo de Alarmar
- `default.png` - Logo por defecto para competidores sin logo

**Formato recomendado:**
- PNG con fondo transparente
- Tamaño: 100x100 píxeles mínimo
- Aspecto cuadrado o ligeramente rectangular
